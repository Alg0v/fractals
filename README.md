# Fractals

Un visualiseur de fractales multithread libre.
Il permet de visualiser les ensembles de Julia ainsi que l'ensemble de Mandelbrot.

## Compiler le projet

```bash
git clone https://github.com/Alg0v/fractals
cd fractals
mkdir build
cd build
cmake ..
cmake --build .
```

### Utilisation

```bash
./fractals
```
